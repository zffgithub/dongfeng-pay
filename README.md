# dongfeng-pay
东风支付系统，采用大家喜欢的go语言编写，前后端齐全（管理后台，商户后台，代理后台，网关，结算，代付，等）
# 目的
接触了很多支付系统，绝大部分都是用java、php写的，本人想着为go献出一份绵薄之力。该开源项目只能用于学习，不准备用于任何的非法商业活动，否则后果自负。
